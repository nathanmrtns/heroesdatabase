# Usage

This project requires `Node >=12` and `NPM >= 6`.

After you have cloned this repo, install the dependencies with:

```
npm install
```

You can then start the application running:

```
npm run dev
```

That's it. Just Access `http://localhost:3000` in your browser.

### Linting

```
npm run lint
```

### Testing

```
npm run test
```
